# Retail-Customer-Segmentation

Project Overview

Retail Customer Segmentation is a data analytics project that groups customers based on their purchasing behavior and demographic information. The objective is to identify different customer segments so that businesses can better understand their customers and create targeted marketing strategies.

# Objective

- Analyze customer data.
- Identify similar customer groups.
- Improve customer engagement and marketing campaigns.
- Support business decision-making through data-driven insights.

# Tools Used

- Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn)
- Excel
- Power BI
- Jupyter Notebook

# Methodology

1. Load and clean the dataset.
2. Perform Exploratory Data Analysis (EDA).
3. Select important features such as Age, Annual Income, and Spending Score.
4. Apply the K-Means clustering algorithm.
5. Visualize customer segments.
6. Build a Power BI dashboard to present key insights.

# Results

The project successfully divides customers into meaningful groups, helping businesses identify high-value customers, loyal customers, and customers with growth potential. These insights can be used to improve marketing strategies, customer retention, and overall business performance.
